Mon projet utilise l'intelligence arificiel pour traduire la langue des signes en temps Reel 
j'ai mis comme fonctionnaliter : Detection des mains via Mediapipe, Pre-traitement des images( Flou Gaussin et conversion Rgb) avec OpenCV, traduction vocal asynchrone(threading) avec pyttsx3, interface visuelle avec trace de mouvement et barre de stabiliter.
